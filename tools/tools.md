# Tools

## Writing, Editing, and Review

- [Grammarly](https://app.grammarly.com/) — Grammar and clarity checker.
- [Hemingway Editor](https://hemingwayapp.com/) — Readability and sentence simplification tool.
- [Datayze Passive Voice Detector](https://datayze.com/passive-voice-detector) — Finds passive voice.
- [Quetext](https://www.quetext.com/) — Plagiarism checker.
- [QuillBot](https://quillbot.com/) — Paraphrasing and rewriting tool.
- [Wordtune](https://www.wordtune.com/) — AI rewriting assistant.
- [Vale](https://vale.sh/docs) — Prose linter for style-guide enforcement.
- [alex](https://alexjs.com/) — Inclusive language linter.
- [textlint](https://textlint.github.io/) — Pluggable natural-language linter.

## Screenshots, Graphics, and Diagrams

- [Canva](https://www.canva.com/) — Visual design tool for article images and simple graphics.
- [Annotely](https://annotely.com/) — Screenshot annotation tool.
- [Mermaid](https://mermaid.js.org/) — Diagrams and flowcharts using text.
- [Excalidraw](https://excalidraw.com/) — Hand-drawn-style diagrams.
- [draw.io / diagrams.net](https://www.diagrams.net/) — Diagrams and flowcharts.
- [Figma](https://www.figma.com/) — Interface design, design systems, and collaborative diagrams.
- [Lucidchart](https://www.lucidchart.com/) — Diagramming and process mapping.
