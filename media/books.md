# Books

- [Open Technical Communication, 2e](https://alg.manifoldapp.org/projects/open-tc-2e) — Free technical communication textbook.
- [Docs for Developers](https://docsfordevelopers.com/) — Book site for *Docs for Developers: An Engineer’s Field Guide to Technical Writing*.
- [Docs for Developers on Amazon](https://www.amazon.com/Docs-Developers-Engineers-Technical-Writing/dp/1484272161) — Purchase page.
- [The Product Is Docs](https://www.amazon.com/Product-Docs-technical-documentation-development/dp/B085KK6G8F) — Essays on product documentation.
- [Every Page Is Page One](https://everypageispageone.com/the-book/) — Topic-based writing for the web.
- [The Mayfield Handbook of Technical and Scientific Writing](https://www.mit.edu/course/21/21.guide/toc.htm) — Classic technical and scientific writing reference.
- [Writing Handbook](https://www.julian.com/guide/write/intro) — Writing guide by Julian Shapiro.
- [Technical Writing 101](https://www.gettextbooks.com/isbn/9780970473363/) — Planning and writing technical content.
- [Write the Docs Book List](https://www.writethedocs.org/books/) — Community-recommended books.
