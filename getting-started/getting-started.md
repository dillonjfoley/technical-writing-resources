# Getting Started

- [Technical Writing Roadmap](https://roadmap.sh/technical-writer) — A skills roadmap for technical writers.
- [Google Technical Writing Courses](https://developers.google.com/tech-writing) — Free courses for improving engineering and developer documentation.
- [Technical Writing One](https://developers.google.com/tech-writing/one) — Fundamentals of clear technical writing.
- [Technical Writing Two](https://developers.google.com/tech-writing/two) — Intermediate topics for structuring larger documents.
- [Write the Docs Software Documentation Guide](https://www.writethedocs.org/guide/) — Community guide to software documentation.
- [The Good Docs Project Templates](https://www.thegooddocsproject.dev/template) — Open documentation templates for common doc types.
- [Diátaxis](https://diataxis.fr/) — A documentation framework based on tutorials, how-to guides, reference, and explanation.
- [Learn API Documentation](https://idratherbewriting.com/learnapidoc/) — API documentation course by Tom Johnson.
- [Open Technical Communication, 2e](https://alg.manifoldapp.org/projects/open-tc-2e) — Free open textbook for technical communication.
- [Markdown Guide](https://www.markdownguide.org/getting-started/) — Beginner-friendly Markdown reference.
