# Podcasts and Video Resources

- [The Not-Boring Tech Writer](https://thenotboringtechwriter.com/) — Technical writing podcast.
- [10-Minute Tech Comm](https://podcasts.apple.com/us/podcast/10-minute-tech-comm/id920575683) — Short interviews with technical communication practitioners.
- [API the Docs Podcast](https://apithedocs.org/podcast?page=1) — API documentation and developer portal podcast.
- [I’d Rather Be Writing YouTube Channel](https://www.youtube.com/@idratherbewriting) — Technical writing and API docs videos.
- [Seeing Your Docs Through Different Eyes](https://www.youtube.com/watch?v=vj7TyUgBEW8) — Mapping documentation user experiences by Aaron Collier.
- [What Do You Look for When Hiring Technical Writers?](https://www.youtube.com/watch?v=tv1DhmUT-Fo) — Hiring-focused technical writing video.
- [Code Samples in Documentation](https://www.youtube.com/watch?v=_gtzUpvGfXY) — Code samples in docs by Brian Murray.
