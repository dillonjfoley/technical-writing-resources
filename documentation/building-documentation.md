# Building Documentation Sites

- [How to Create a Documentation Site from Scratch](https://medium.com/empathyco/how-to-create-a-documentation-site-from-scratch-3ce54bf0257) — Tutorial by José A. Blanco.
- [How to Build a Docs Site](https://dev.to/michaelcurrin/how-to-build-a-docs-site-4h42) — Documentation site guide by Michael Currin.
- [Docusaurus](https://docusaurus.io/docs) — React-based documentation site generator.
- [MkDocs](https://www.mkdocs.org/) — Markdown-based documentation site generator.
- [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) — Popular theme and tooling layer for MkDocs.
- [Sphinx](https://www.sphinx-doc.org/) — Documentation generator commonly used for Python and technical projects.
- [Read the Docs](https://docs.readthedocs.com/) — Documentation hosting and build platform.
- [Antora](https://docs.antora.org/antora/latest/) — AsciiDoc-based documentation site generator.
- [Hugo Documentation](https://gohugo.io/documentation/) — Fast static-site generator.
- [Jekyll Documentation](https://jekyllrb.com/docs/) — Static site generator often used with GitHub Pages.
- [Just the Docs](https://just-the-docs.com/) — Documentation theme for Jekyll and GitHub Pages.
