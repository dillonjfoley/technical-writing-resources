# Courses and Learning Platforms

- [Tech Writers Master Class](https://www.udemy.com/course/tech-writers-master-class/) — Udemy course by Janet Underwood.

- [edX](https://www.edx.org/) — Online courses in writing, computer science, and communication.
- [Google Digital Skills](https://learndigital.withgoogle.com/digitalskills/courses) — Digital skills courses.
- [Microsoft Learn](https://learn.microsoft.com/) — Microsoft technical training and documentation examples.
- [GitHub Skills](https://skills.github.com/) — GitHub and Git learning modules.
