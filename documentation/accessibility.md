# Accessibility, Plain Language, and Inclusive Writing

- [Web Content Accessibility Guidelines](https://www.w3.org/TR/WCAG21/) — WCAG 2.1 standard.
- [Digital.gov Plain Language Guide](https://digital.gov/guides/plain-language) — U.S. plain language guidance.
- [PlainLanguage.gov Law and Requirements](https://www.plainlanguage.gov/law/) — Plain Writing Act background and requirements.
- [Google: Write Inclusive Documentation](https://developers.google.com/style/inclusive-documentation) — Inclusive developer documentation guidance.
- [Apple Human Interface Guidelines: Inclusion](https://developer.apple.com/design/human-interface-guidelines/inclusion) — Inclusive product language.
- [Academy Software Foundation Inclusive Language Guide](https://www.aswf.io/inclusive-language-guide/) — Inclusive language guidance for software projects.
- [Americans with Disabilities Act National Network](https://adata.org/learn-about-ada) — ADA learning resources.
- [Witty Works](https://www.witty.works/) — Inclusive language tool and resources.
