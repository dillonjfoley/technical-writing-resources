# API Documentation

## Learn API Documentation

- [Documenting APIs](https://idratherbewriting.com/learnapidoc/) — Comprehensive API documentation course.
- [Getting Started in API Documentation Writing](https://writeonce.org/starting_as_a_programmer_writer.php) — Guide by Robert Delwood.
- [How to Read API Documentation](https://www.departmentofproduct.com/blog/how-to-read-api-documentation/) — Useful for product managers, writers, and beginners.
- [Working with APIs](https://support.knowledgeowl.com/help/work-with-apis) — KnowledgeOwl API basics.
- [API the Docs](https://apithedocs.org/) — API documentation conference, talks, podcast, and community resources.

## OpenAPI, Swagger, and API Specs

- [OpenAPI Specification](https://swagger.io/specification/) — Official OpenAPI specification page.
- [OpenAPI Initiative Learning Resources](https://learn.openapis.org/) — Learn how to write and use OpenAPI descriptions.
- [Swagger Documentation](https://swagger.io/docs/) — Swagger Editor, Swagger UI, and OpenAPI tooling docs.
- [Postman: Add API Documentation](https://learning.postman.com/docs/publishing-your-api/documenting-your-api/) — How to create API documentation in Postman.
- [Postman Learning Center](https://learning.postman.com/) — Official Postman documentation.
- [Postman Academy](https://academy.postman.com/) — Hands-on API courses.
- [Redocly API Docs](https://redocly.com/docs/cli/api-docs) — API reference documentation from OpenAPI.
- [Scalar Docs](https://scalar.com/products/docs/getting-started) — Modern API documentation based on OpenAPI.
- [Bump.sh Documentation](https://docs.bump.sh/help/) — API documentation and changelog publishing.
- [Fern Documentation](https://buildwithfern.com/learn/home) — Generate docs and SDKs from API specs.
- [Mintlify Documentation](https://www.mintlify.com/docs) — Documentation platform with API reference support.

## API Clients and Testing Tools

- [Postman](https://www.postman.com/product/) — API platform for building, testing, documenting, and monitoring APIs.
- [Insomnia](https://insomnia.rest/) — API client for REST, GraphQL, WebSockets, gRPC, and more.
- [Hoppscotch](https://docs.hoppscotch.io/) — Open-source API development tool.
- [Apidog Docs](https://docs.apidog.com/) — API design, testing, and documentation platform docs.
- [Stoplight Docs](https://docs.stoplight.io/) — API design and documentation platform docs.
